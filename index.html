MeerVR Garden Experience

We started this project sketching a plan of what the user would do and learn on the website. Using html, javascript, jQuery and aframe we built a webvr application for users to learn more about gardening. 3D models were made using blender. To make sure that users had a good experience on the website we used the google cardboard design rules and added our own use cases on the go.
